1搭建虚拟机环境

理清思路：

1.1本地windows ->1.2dns解析为主机ip ->1.3检查防火墙tcp端口22是否放行->1.4服务器ubuntu openssh-server启动->ssh会话建立

实操：
ping 主机.mshome.net（mshome.net是hyper-v虚拟机的域名）
再ssh 用户名@你的主机名.mshome.net or ssh 用户名@172.

检查：防火墙sudo ufw allow ssh
sudo ufw status和ssh服务sudo apt update
sudo apt install openssh-server
sudo systemctl enable ssh
sudo systemctl start ssh

2免密登录，保存登录信息

本地生成密钥对——登录虚拟机——创建ssh目录——本地复制公钥到虚拟机——测试免密登录
ssh-keygen -t rsa -b 4096 -C “你的邮箱”
ssh jimmy@172.24.49.199
mkdir -p ~/.ssh
chmod 700 ~/.ssh
type C:\Users\JimmyHuang.ssh\id_ed25519.pub | ssh jimmy@172.24.49.199 “cat >> ~/.ssh/authorized_keys && chmod 600 ~/.ssh/authorized_keys”
ls -l ~/.ssh
cat ~/.ssh/authorized_keys
ssh -i C:\Users\JimmyHuang.ssh\id_ed25519 jimmy@172.24.49.199

3启动mysql服务

安装——启动——配置——初始化&授权——建db建表插数据
1sudo apt update
sudo apt install mysql-server
2sudo systemctl start mysql
sudo mysql -u root -p
3.1 检查配置文件
sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf
bind-address = 0.0.0.0
sudo systemctl restart mysql
3.2防火墙
sudo ufw allow 3306(这里打开vm的fwd，不用打开宿主机的fw，因为用的是localhost访问mysql服务)
sudo ufw reload
4CREATE DATABASE scout;
只授予localhost权限即只能从vm访问db！！！CREATE USER ‘jimmy‘@’localhost’ IDENTIFIED BY ‘你的密码’;GRANT ALL PRIVILEGES ON scout.* TO ‘jimmy‘@’localhost’;
要允许任意ip访问！！！create user ‘jimmy‘@’%’ identified by ‘你的密码’;
grant all privileges on scout.* to ‘jimmy‘@’%’;

FLUSH PRIVILEGES;
5mysql -u root -p scout
create table scout(
id int auto_increment primary key,
actor varchar(255)
)
insert into scout(actor) values (‘jimmy’)；引号！！！
